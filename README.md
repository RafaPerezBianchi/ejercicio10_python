# Ejercicio 10

## Opción 1: Con Anaconda (recomendado)
1. Instalar Anaconda: https://www.anaconda.com/download
2. Abrir Anaconda Prompt
3. Navegar a la carpeta del proyecto:
   cd ruta/del/proyecto
4. Ejecutar:
   jupyter notebook
5. Abrir la carpeta notebooks y ejecutar ejercicio10.ipynb

## Opción 2: Sin Anaconda
1. Instalar Python: https://www.python.org/downloads
2. Instalar Jupyter desde la terminal:
   pip install notebook
3. Navegar a la carpeta del proyecto:
   cd ruta/del/proyecto
4. Ejecutar:
   jupyter notebook
5. Abrir la carpeta notebooks y ejecutar ejercicio10.ipynb
